# Building Permits

Building permit data for Fairfax County development activity.

## Data Source

Fairfax County building permit records.
Check Fairfax County GIS Open Data Portal for available datasets.

## Directory Structure

- `raw/` - Original downloads (LOCAL ONLY - not committed)
- `processed/` - Optimized Parquet files (committed to GitHub)

## Use Cases

- Track new construction activity
- Identify renovation patterns
- Analyze development trends by area
- Estimate property improvements

## Processing Workflow

See `raw/README.md` for download and processing instructions.
