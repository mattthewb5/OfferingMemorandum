# Crime Data

Public safety and crime incident data for Fairfax County.

## Data Source

Fairfax County Police Department crime reports.
Check county open data portal for available datasets.

## Directory Structure

- `raw/` - Original downloads (LOCAL ONLY - not committed)
- `processed/` - Optimized Parquet files (committed to GitHub)

## Use Cases

- Neighborhood safety analysis
- Crime trend monitoring
- Property value impact assessment
- Community comparison

## Privacy Notes

Crime data may be generalized to block level to protect privacy.
Individual incident details are typically aggregated.
