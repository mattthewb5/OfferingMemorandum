# Schools Data

School performance and information for Fairfax County Public Schools (FCPS).

## Data Source

- **Virginia Department of Education**: https://schoolquality.virginia.gov/
- **FCPS Website**: https://www.fcps.edu/

## Directory Structure

- `performance/` - Test scores, ratings, enrollment data

## FCPS Overview

Fairfax County Public Schools is the 10th largest school district in the US:
- ~190,000 students
- 200+ schools
- Strong academic reputation

## Data Types

- SOL (Standards of Learning) pass rates
- Accreditation status
- Enrollment numbers
- Demographic information
- Advanced program participation
