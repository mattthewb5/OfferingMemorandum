# Transit Data

Public transportation data for Fairfax County area.

## Data Sources

- **WMATA (Metro)**: Washington Metropolitan Area Transit Authority
- **Fairfax Connector**: Fairfax County bus system
- **VRE**: Virginia Railway Express

## Directory Structure

- `metro/` - Metrorail stations and lines

## Transit Context

Fairfax County is served by:
- **Silver Line**: Connects Tysons, Reston to DC
- **Orange Line**: Extends to Vienna
- **Blue Line**: Near southern areas
- **Fairfax Connector**: Extensive bus network
- **VRE**: Commuter rail to DC
